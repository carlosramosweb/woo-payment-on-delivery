Receba em dinheiro, cheque, no cartão de crédito, débito, cartão alimentação (voucher) e MultiBanco no ato da entrega.

== Description ==

Receba em dinheiro no ato da entrega e ainda com informação se o cliente precisará de troco, aceite também se quiser pagamento no cheque, no cartão de crédito, débito, cartão alimentação (voucher) e MultiBanco.

Adicione (Gateway) novas formas de pagamentos em dinheiro, cheque, cartão de crédito, débito, alimentação ou MultiBanco tudo isso no ato da entrega. A ideal seria entregas expressas com Motoboy ou transporte próprio. As empresas de FastFood, Pizzarias e similares também podem tirar proveito desse ótimo plugin.

Credit icon: Flaticon = https://www.flaticon.com/

= Recomendamos =
* Se você precisa imprimir na fatura o valor de troco ou qual será o cartão use esse plugin modificado abaixo.
* Download do plugin modificado para impressão do troco ou qual cartão: http://criacaocriativa.com/plugins/woocommerce-delivery-notes.zip
* è um plugin modificado de terceiro então não terá suporte para erros ou desatualização.

= Recursos =
* Disponibiliza campo para o cliente informar o valor de troco que precisa (para o entregador levar).
* A informação de troco, tipo de cartão ou cheque fica impresso na Notas do Pedido.
* Aceita Entregas Nativas do WooCommerce (Retirar no local, Taxa fixa e Frete grátis).
* Aceita Correios Brasil (WooCommerce Correios).
* Aceita Entregas Avançadas (Woocommerce Advanced Shipping).
* O sistema aceita todos os pagamentos ou somente um.

== Installation ==

= Instalação =
1. Fazer upload do plugin para o diretório [www.seusite.com.br/wp-content/plugins/].
2. Ative o plugin através do menu lateral do painel administrativo do WordPress [Plugins].
3. Vá no menu [WooCommerce > Configurações > Pagamentos] na opção 'Na Entrega' ou 'Delivery' e clique em Habilitado.
4. Clique no item em lista do plugin [nome do plugin] e aplique qual o método de entrega será ativo para o plugin funcionar no campo 'Ativar métodos de entrega'. É importante que tenha escolhido um metodo de entrega para o plugin ser exibido no checkout.
